## 使用建议

- **第一轮**：按 `leetcode_interview_guide.md` 的分类刷题，配合 `problems/` 中的题目实战
- **第二轮**：对照 `interview_prep_dl_pytorch.md` 复习 DL/PyTorch 核心概念
- **第三轮（OKX 专项）**：精读 `ai_interview/okx_interview_prep.md`，重点攻克行为序列建模、生成式推荐、因果推断三个模块
- **考前冲刺**：重点复习薄弱章节，熟记高频模板
