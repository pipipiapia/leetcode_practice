# 01_collision：对撞指针

> 两个指针从数组两端向中间移动，常用于有序数组

## 核心特征

- 数组必须有序（或可通过排序转化为有序）
- 左指针从 0 开始，右指针从末尾开始
- 两指针相向移动，根据条件决定移动哪一侧

## 解题框架

```python
left, right = 0, len(nums) - 1
while left < right:
    if 条件满足:
        记录答案
        left += 1   # 或 right -= 1
    elif nums[left] + nums[right] < target:
        left += 1
    else:
        right -= 1
```

## 关键思想

**为什么移动较矮侧能保证不漏解？**
以盛水问题为例：面积受限于较矮的那根柱子。移动较高的柱子，新的较矮侧不会比原来高，面积只会更小或不变。因此移动较矮侧才有可能找到更大的解。
