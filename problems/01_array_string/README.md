# Category 1：数组 & 双指针 & 滑动窗口

> 面试占比约 20%，最高优先级

## 核心知识点

**双指针（Two Pointers）** 是一种通用技巧，核心是用两个指针遍历数组/链表，根据问题特征演化为三种形态：

| 类型 | 指针方向 | 适用场景 | 关键特征 |
|------|----------|----------|----------|
| 对撞指针 | 两端向中间 | 有序数组求和、容器盛水 | left + right 向 target 逼近 |
| 快慢指针 | 同向而行 | 链表判环、数组去重 | fast 扫描，slow 记录结果 |
| 滑动窗口 | 两端同向滑动 | 子数组/子串问题 | 窗口扩缩，O(n) 解决 |
