# 滑动窗口（Sliding Window）

> 通过维护一个窗口（连续区间），在 O(n) 时间内解决子数组/子串问题

## 核心分类

### 固定窗口大小
窗口长度 k 固定，右指针扩展到 k 后，整体向右滑动。

```python
# 先计算第一个窗口
window_sum = sum(nums[:k])
max_avg = window_sum / k
# 滑动窗口
for i in range(k, len(nums)):
    window_sum += nums[i] - nums[i - k]
    max_avg = max(max_avg, window_sum / k)
```

### 可变窗口大小
窗口大小随条件动态变化，常配合哈希表使用。

```python
left = 0
for right in range(len(nums)):
    # 扩展窗口
    while 满足收缩条件:
        # 更新答案
        # 收缩窗口
        left += 1
```

## 常见变体

| 场景 | 窗口类型 | 技巧 |
|------|----------|------|
| 固定长度子数组求和/平均值 | 固定 | 先初始化第一个窗口 |
| 子数组最大/最小平均值 | 固定 | 滑动时加减边界 |
| 无重复字符最长子串 | 可变 | 右扩左缩，哈希表记录位置 |
| 最小覆盖子串 | 可变 | 哈希表计数，两个指针维护覆盖状态 |
| 和>=target最小子数组 | 可变 | 右扩左缩，维护最短长度 |
| 字母异位词/排列 | 固定 | 字符计数数组比较 |
