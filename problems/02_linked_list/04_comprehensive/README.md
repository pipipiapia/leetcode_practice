## 关键思想

**重排链表 = 找中点 + 反转后半 + 交替合并**
三步走：快慢指针找中点，反转后半段，两个子链表交替合并。

**LRU 缓存 = 哈希表 + 双向链表**
哈希表 O(1) 查找，双向链表 O(1) 移动到头部/删除尾部。访问时移到头部，容量满时删尾部。
