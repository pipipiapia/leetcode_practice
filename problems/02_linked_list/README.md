# Category 2：链表

> 面试占比约 8%，字节跳动极高频

## 核心知识点

- 快慢指针：Floyd 判圈算法（慢1步/快2步→相遇→一指针回头→再走→入环点）
- 反转操作：三指针 prev/curr/next 迭代，或递归传递
- 哑节点技巧：统一处理头节点特殊情况
