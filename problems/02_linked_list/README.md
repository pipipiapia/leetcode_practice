# 02_linked_list：链表

> 链表的核心操作：虚拟头节点、反转、快慢指针、综合题

## 核心特征

- 链表不能随机访问，必须从头遍历
- 插入/删除 O(1)（已知前驱节点时），查找 O(n)
- Python 中用 `ListNode(val, next)` 表示
