# Category 9：图 & 并查集

## 核心知识点

- BFS：队列 + 层次计数，求最短路
- 拓扑排序（Kahn）：构建入度表 → 入度0入队 → BFS
- 并查集：find() 路径压缩 + union() 按秩合并
