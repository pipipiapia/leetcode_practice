# 02_monotonic：单调栈

> 栈内元素保持单调递增或递减，用于找"下一个更大/更小元素"

## 核心特征

- 找"下一个更大"→ 单调递减栈（栈顶最小）
- 找"下一个更小"→ 单调递增栈（栈顶最大）
- 每个元素最多入栈出栈各一次，O(n)

## 解题框架

```python
# 找下一个更大元素
stack = []          # 存下标
result = [-1] * n
for i in range(n):
    while stack and nums[stack[-1]] < nums[i]:
        result[stack.pop()] = nums[i]
    stack.append(i)
```

## 关键思想

**单调栈的本质？**
维护一个"待处理"的候选序列。新元素到来时，比它小的都出栈（找到了下一个更大），比它大的保留（还在等更大的）。
