# 02_two_dim：二维动态规划

> 状态为二维数组的 DP 问题，常见于双字符串或网格

## 解题框架

```python
# 二维 DP 模板
dp = [[0] * (m + 1) for _ in range(n + 1)]
for i in range(1, n + 1):
    for j in range(1, m + 1):
        dp[i][j] = 转移方程
return dp[n][m]
```

## 关键思想

**编辑距离：** dp[i][j] = word1[0..i] 和 word2[0..j] 的最小编辑次数。三选一：删除、插入、替换。

**LCS：** 字符相同则 dp[i][j] = dp[i-1][j-1] + 1，不同则取 max(dp[i-1][j], dp[i][j-1])。
