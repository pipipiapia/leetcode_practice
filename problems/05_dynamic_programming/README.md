# 05_dynamic_programming：动态规划

> 将问题拆成重叠子问题，记忆化避免重复计算

## 子目录

| 目录 | 英文含义 | 考点 |
|------|----------|------|
| 01_one_dim | 一维 DP | 子数组、爬楼梯、打家劫舍、背包 |
| 02_two_dim | 二维 DP | 路径、编辑距离、LCS |
| 03_subsequence | 子序列 | 最长递增子序列 |

## 核心特征

- 三要素：状态定义、转移方程、初始条件
- 先想"最后一步"，再推"前一步怎么来"
- 与回溯的区别：DP 解决重叠子问题，回溯解决排列组合
