## 核心知识点

- 遍历框架：前/中/后序（递归+迭代）、BFS 层序，必须双写
- 路径问题：后序 DFS 回溯，全局变量维护最大值
- BST 性质：中序有序性用于验证/查找，公共祖先单侧剪枝
