# 04_binary_tree：二叉树

> 二叉树的遍历、BST、构造、路径问题

## 子目录

| 目录 | 英文含义 | 考点 |
|------|----------|------|
| 01_traversal | 遍历 | 层序、锯齿层序、深度、右视图 |
| 02_bst | 二叉搜索树 | 验证 BST |
| 03_construction | 构造 | 前序+中序建树、展开为链表、序列化 |
| 04_path_lca | 路径与LCA | 路径总和、最近公共祖先 |

## 核心特征

- 递归是二叉树的灵魂（遍历、构造、路径都靠递归）
- BST 的中序遍历是升序序列
- 序列化/反序列化考察树的编解码
