# Category 6：栈 & 单调栈

## 核心知识点

- 单调栈模板：`遍历 → 栈顶比当前元素大/小时弹出结算 → 当前元素入栈`
- 括号匹配：左括号入栈，右括号出栈验证
- 单调队列：滑动窗口最大值（双端队列）
